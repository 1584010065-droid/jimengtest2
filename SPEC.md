# 图像生成效果优化 Bot MVP - 规范文档

## 1. 项目概述

- **项目名称**: 图像生成效果优化 Bot
- **项目类型**: Web 应用 (Next.js + React + Tailwind CSS)
- **核心功能**: AI 图像生成效果分析与优化对话助手
- **目标用户**: 普通用户、设计师（AI 图像生成爱好者）

## 2. UI/UX 规范

### 2.1 布局结构

```
┌─────────────────────────────────────────────────────────┐
│  顶部导航栏 (header)                                     │
│  [Logo + 标题]                    [+ 新对话]            │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  主内容区 (main)                                         │
│  - 空状态: Logo + 建议列表                               │
│  - 对话区: 用户/机器人消息气泡                            │
│  - 分析卡片: 场景标签 + 分析结果 + 解决方案 + 优化提示词   │
│                                                         │
├─────────────────────────────────────────────────────────┤
│  底部输入区 (footer)                                     │
│  [📷] [输入框...                    ] [⚙️] [发送]        │
└─────────────────────────────────────────────────────────┘
```

### 2.2 响应式断点

- 移动端: < 640px
- 平板: 640px - 1024px
- 桌面: > 1024px

### 2.3 视觉设计

#### 颜色系统

```css
--bg-color: #F8F8F9;           /* 背景色 */
--card-bg: #FFFFFF;             /* 卡片背景 */
--text-main: #333333;           /* 主文字 */
--text-muted: #888888;          /* 辅助文字 */
--primary-color: #F27A33;       /* 主色调 - 橙色 */
--border-color: #EAEAEA;        /* 边框色 */
--shadow-sm: 0 2px 8px rgba(0, 0, 0, 0.03);
--shadow-md: 0 4px 24px rgba(0, 0, 0, 0.06);
```

#### 字体

- 主字体: -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto
- 标题: 16px (导航), 15px (消息)
- 正文: 14px
- 代码/提示词: monospace, 14px

#### 间距

- 导航内边距: 16px 24px
- 主内容区内边距: 24px 10%
- 消息间距: 32px
- 卡片内边距: 16px 20px

#### 圆角

- 药丸形: 40px (输入框)
- 卡片: 16px
- 按钮: 20px, 50%

### 2.4 组件规范

#### 顶部导航栏

- 高度: 自适应
- 背景: var(--bg-color)
- 左侧: Logo SVG + 标题文字
- 右侧: "新对话" 按钮 (边框样式)

#### 空状态

- 居中布局
- 大 Logo (56x56)
- 建议列表 (3个建议项)
- 建议项: 白色卡片 + hover 阴影效果

#### 消息气泡

- 用户消息: 右侧, 背景 primary-color, 白色文字
- 机器人消息: 左侧, 白色背景, 文字 #333
- 头像: 36x36 圆形, 带阴影
- 最大宽度: 800px

#### 分析结果卡片

- 场景标签: 橙色背景 (10%透明度), 橙色文字, 圆角 12px
- 卡片分区: 底部边框分隔
- 提示词框: 灰色背景 (#F4F4F5), 等宽字体
- 复制按钮: 右上角定位

#### 底部输入框

- 药丸形状容器
- 图片上传按钮 (左侧)
- 文本输入框 (中间, 可扩展)
- 参数设置按钮
- 发送按钮 (圆形, 黑色背景)

## 3. 功能规范

### 3.1 核心功能

1. **新对话**: 清除当前对话，显示空状态
2. **图像上传**: 支持上传参考图、生成图 (模拟)
3. **提示词输入**: 文本输入框
4. **AI 分析回复**: 模拟显示预设分析结果
5. **复制功能**: 一键复制优化后的提示词

### 3.2 用户交互流程

1. 用户进入页面 → 显示空状态 + 建议列表
2. 用户点击建议项 → 发送预设消息 → 显示模拟分析结果
3. 用户可输入自定义提示词 → 模拟分析回复
4. 点击"新对话" → 重置对话

### 3.3 模拟数据

提供3个预设场景的模拟回复：

1. **风格转绘**: 图标→糖果材质
2. **主体替换**: 猫→男孩+羊群
3. **同风格改编**: 东京塔→东方明珠海报
